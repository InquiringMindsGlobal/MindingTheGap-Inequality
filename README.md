These python packages are meant for Minding The Gap article on Medium Science Spectrum publication: https://medium.com/p/109a77e6de1b/edit

1. Inequality_Simulation_Luck_Based: code simulation of stage 1: Luck-based inequality
2. Inequality_Simulation_Meritocracy_Based: code simulation of stage 2: Meritocracy-based inequality
3. Inequality_Simulation_Power_Hierarchy_Based: code simulation of stage 3: Power-based inequality
4. Inequality_Gini_Evolutions: Comparison plot of Gini evolution between the three stages 
5. Gini_comparison with vs without structural hierarchy: comparison between two types of inequality structures
6. Gini_comparison between 3 variations of longevity: comparison between three types of inequality structures
